Privacy Policy

Last updated: 2026/02/28

1. Data Collection

This extension does not collect, store, transmit, or share any personal information or user data. All functionality operates locally within the user's browser.

2. Authentication Tokens

If the user chooses to enter a Discord authentication token, the token is used locally within the browser session to perform the requested action. The extension does not permanently store the token and does not transmit it to any external server.

3. Third-Party Services

This extension interacts only with the official Discord website (https://discord.com) as required for its functionality. No information is sent to the developer or any third party.

4. Data Security

All operations occur locally within the user's browser environment. The developer does not have access to any user data.

5. Changes to This Policy

This Privacy Policy may be updated if the extension functionality changes. Any updates will be reflected on this page.
